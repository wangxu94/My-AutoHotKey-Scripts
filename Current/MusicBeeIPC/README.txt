MusicBeeIPC SDK for AutoHotkey
==============================

Author: Kerli Low

Email: kerlilow@gmail.com

Homepage: http://www.zorexxlkl.com/musicbeeipc

Description: Control MusicBee with AutoHotkey using the MusicBeeIPC plugin.

Installation:
    Include the MusicBeeIPC.ahk file as follows:
        #include MusicBeeIPC ; the path to the MusicBeeIPC SDK
        #include MusicBeeIPC.ahk
